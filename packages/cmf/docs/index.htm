React-CMF

A framework built on top of best react libraries

Why

React itself is a library. There is an ecosystem you must learn to start writing apps. We have constraints to follow and this framework is the response to theses contraints:

  • App can be extended
  • View can be customized without changing code

What

This is the guidelines we follow:

  • Settings comes from the server
  • Code only configurable React components
  • Create content type to handle common actions everywhere

How

A registry let's us use IoC pattern so configuration is all about create relation between components.

  • Provide a component registry
  • Provide a yeoman generator
  • Provide a set of reusable components
  • Provide documentation
